# Daten mit Veeam wiederherstellen

## Ziel

Kurze Checkliste zur Wiederherstellung von Daten aus einem Veeam-Backup.

## Checkliste

- Betroffenen Benutzer, Datei, Ordner oder Postfach klären.
- Benötigten Wiederherstellungszeitpunkt klären.
- Prüfen, ob der Wiederherstellungspunkt noch vorhanden ist.
- Freigabe zur Wiederherstellung einholen.
- Veeam Backup Server öffnen.
- Passenden Restore-Typ auswählen.
- Gewünschtes Backup und Datenquelle auswählen.
- Gesuchte Datei oder E-Mail auswählen.
- Wiederherstellung durchführen.
- Prüfen, ob die richtigen Daten am richtigen Ziel wiederhergestellt wurden.
- Sicherstellen, dass keine bestehenden Daten versehentlich überschrieben wurden.
