# Kaspersky Endpoint Security installieren

## Ziel

Diese Anleitung beschreibt die grundlegende Installation einer Endpoint-Security-Lösung am Beispiel von Kaspersky Endpoint Security.

## Ausgangslage

Ein Arbeitsplatz soll mit einer Endpoint-Security-Lösung ausgestattet oder auf eine aktuelle Version gebracht werden. Vor der Installation muss geprüft werden, ob bereits eine alte Version installiert ist.

## Vorbereitung

Vor der Installation prüfen:

- Ist bereits eine ältere Version installiert?
- Gibt es eine Unternehmenslizenz?
- Gibt es ein zentrales Installationspaket?
- Muss das Gerät vorher neu gestartet werden?
- Gibt es Vorgaben, welche Module installiert werden sollen?

## Alte Version prüfen

1. Systemsteuerung öffnen.
2. Installierte Programme prüfen.
3. Vorhandene alte Version dokumentieren.
4. Alte Version nach Vorgabe deinstallieren.
5. Gerät neu starten, falls erforderlich.

## Installation

1. Installationspaket aus vertrauenswürdiger Quelle verwenden.
2. Setup starten.
3. Installationsoptionen nach Unternehmensvorgabe auswählen.
4. Nicht benötigte Module nur entfernen, wenn dies dokumentiert und freigegeben ist.
5. Installation abschließen.
6. Gerät neu starten, falls erforderlich.

## Module und Vorgaben

Welche Schutzmodule installiert werden, hängt von der Unternehmensvorgabe ab. Module sollten nicht ohne Freigabe entfernt werden.

## Nachkontrolle

Prüfen:

- Schutzstatus aktiv
- Lizenz aktiv
- Signaturen aktuell
- zentrale Verwaltung erreichbar
- keine Warnmeldungen im Client

## Wichtiger Hinweis

Zugangsdaten, Lizenzpfade, interne Servernamen und Kundendaten gehören nicht in eine öffentliche Dokumentation.

## Dokumentation im Ticket

Festhalten:

- Gerät
- alte Version entfernt: ja/nein
- neue Version installiert
- Lizenz aktiv: ja/nein
- Neustart durchgeführt
- Schutzstatus geprüft
