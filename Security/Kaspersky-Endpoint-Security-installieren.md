# Kaspersky Endpoint Security installieren

## Ziel

Kurze Checkliste zur Installation von Kaspersky Endpoint Security.

## Checkliste

- Prüfen, ob bereits eine alte Version installiert ist.
- Unternehmenslizenz und freigegebenes Installationspaket prüfen.
- Vorgaben zu Modulen und Einstellungen beachten.
- Alte Version nach Vorgabe deinstallieren.
- Gerät neu starten, falls erforderlich.
- Installationspaket aus vertrauenswürdiger Quelle starten.
- Module nur nach Freigabe ändern.
- Installation abschließen und Gerät bei Bedarf neu starten.
- Schutzstatus, Lizenz, Signaturen und zentrale Verwaltung prüfen.
- Keine Zugangsdaten, Lizenzpfade oder interne Servernamen öffentlich dokumentieren.
