# Zertifikat über MMC anfordern

## Ziel

Kurze Checkliste zum Anfordern eines Zertifikats über die MMC.

## Checkliste

- `mmc.exe` öffnen.
- Snap-in `Certificates` hinzufügen.
- Passenden Speicher auswählen, meist `Computer account`.
- `Local computer` auswählen.
- Unter `Personal` den Assistenten `Request New Certificate` starten.
- Verfügbare Zertifikatvorlagen laden.
- Gewünschte Vorlage auswählen.
- Falls die Vorlage fehlt, Berechtigungen und Veröffentlichung prüfen.
- `Enroll` ausführen.
- Zertifikat unter `Personal > Certificates` prüfen.
- Subject, Key Usage, Enhanced Key Usage und Zertifikatpfad kontrollieren.
