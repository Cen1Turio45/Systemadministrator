# File-Shares und Berechtigungen

## Ziel

Kurze Checkliste zum Erstellen und Prüfen von File-Shares.

## Checkliste

- Ordner auf dem Fileserver erstellen.
- Erweiterte Freigabe aktivieren und Freigabename setzen.
- Share-Berechtigungen prüfen.
- NTFS-Berechtigungen prüfen.
- Zugriff nach Möglichkeit über AD-Gruppen steuern.
- Benutzer zur passenden Berechtigungsgruppe hinzufügen.
- Benutzer abmelden oder Gerät neu starten lassen, damit Gruppenmitgliedschaften greifen.
- Zugriff am Client mit `\\SERVERNAME\Freigabename` testen.
- Bei Problemen `Effective Access` prüfen.
- Bei Zugriff verweigert Share- und NTFS-Berechtigungen vergleichen.
