# Active Directory: Benutzer, Gruppen und Domainbeitritt

## Ziel

Kurze Checkliste für typische Active-Directory-Aufgaben im Support.

## Checkliste Domainbeitritt

- Prüfen, ob der Computername bereits in Active Directory existiert.
- DNS des Clients auf den Domain Controller setzen.
- Netzwerkverbindung zum Domain Controller prüfen.
- Uhrzeit und Zeitzone prüfen.
- Domänenbeitritt über Windows-Einstellungen starten.
- Domänenname und berechtigte Zugangsdaten eingeben.
- Gerät neu starten.
- Computerkonto in Active Directory prüfen.

## Checkliste Benutzer und Gruppen

- Benutzer in der passenden OU anlegen.
- Anmeldename, Anzeigename und Pflichtfelder eintragen.
- Temporäres Passwort setzen.
- Passwortänderung bei erster Anmeldung aktivieren.
- Gruppenmitgliedschaften nach Rolle hinzufügen.
- Passwort bei Bedarf zurücksetzen und sicher übergeben.
- Gesperrten Account im Tab `Account` entsperren.
- Account bei Austritt deaktivieren.
- Berechtigungen möglichst über Gruppen vergeben, nicht direkt auf Benutzer.
