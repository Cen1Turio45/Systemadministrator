# Microsoft 365 Admin Center Grundlagen

## Ziel

Diese Anleitung sammelt typische Einstiegsaufgaben im Microsoft 365 Admin Center.

## Grundregel

Bei produktiven Microsoft-365-Änderungen immer die offizielle Microsoft-Dokumentation und interne Unternehmensvorgaben beachten. Lizenznamen und Funktionen können sich ändern.

## Benutzer verwalten

Pfad:

```text
Microsoft 365 Admin Center > Users > Active users
```

Typische Aufgaben:

- Benutzer erstellen
- Benutzer bearbeiten
- Lizenz zuweisen oder entfernen
- Passwort zurücksetzen
- Anmeldestatus prüfen

## Mehrere Benutzer anlegen

Über `Add multiple users` können Benutzer per CSV-Datei erstellt werden. Vorher prüfen:

- Namensschema
- UPN / E-Mail-Adresse
- Lizenz
- Standort
- Gruppenmitgliedschaften

## Domains verwalten

Pfad:

```text
Settings > Domains
```

Typische Aufgaben:

- Domain hinzufügen
- DNS-Einträge prüfen
- Verifizierungsstatus kontrollieren

## Gruppen

Pfad:

```text
Teams & groups
```

Wichtige Gruppentypen:

| Typ | Zweck |
| --- | --- |
| Microsoft 365 Group | Zusammenarbeit mit Mailbox, Kalender und Dateien |
| Distribution List | E-Mail-Verteiler |
| Security Group | Berechtigungen und Zugriff |

## Support-Ticket bei Microsoft erstellen

Pfad:

```text
Support > Help & support
```

Ein gutes Support-Ticket enthält:

- betroffener Dienst
- Zeitpunkt
- betroffene Benutzer
- Fehlermeldung
- bereits geprüfte Schritte
- Screenshots oder Logs

## Service Health prüfen

Pfad:

```text
Health > Service health
```

Damit lässt sich prüfen, ob ein Problem intern verursacht wird oder ob eine Microsoft-Störung vorliegt.

## Exchange Admin Center

Typische Aufgaben:

- geteilte Postfächer erstellen
- Kontakte verwalten
- Mailflow-Regeln prüfen
- einfache Blockierungen oder Weiterleitungen nachvollziehen

## SharePoint und OneDrive

Typische Aufgaben:

- Speicher und Limits prüfen
- Benachrichtigungen konfigurieren
- Sites verwalten
- OneDrive-Einstellungen kontrollieren

## Teams Admin Center

Typische Aufgaben:

- Teams verwalten
- Richtlinien prüfen
- Benutzer- und Meeting-Einstellungen nachvollziehen

## Räume und Equipment

Pfad:

```text
Resources > Rooms & equipment
```

Typische Nutzung:

- Besprechungsräume erstellen
- Geräte wie Projektoren oder Laptops buchbar machen
- Kalender- und Buchungseinstellungen prüfen
