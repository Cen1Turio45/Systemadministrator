# Windows-11-Troubleshooting

## Ziel

Kurze Checkliste für typische Windows-11-Supportfälle.

## Checkliste

- Seit wann besteht das Problem?
- Ist ein Benutzer, ein Gerät oder mehrere Benutzer betroffen?
- Wurde vorher etwas geändert, z. B. Update, Treiber, Software oder Hardware?
- Gerät neu starten und Problem erneut testen.
- Test mit anderem Benutzerprofil durchführen, falls sinnvoll.
- Task-Manager auf CPU, RAM, Datenträger und Netzwerk prüfen.
- Geräte-Manager auf Warnsymbole prüfen.
- Windows Update-Verlauf und Ereignisanzeige prüfen.
- Zuverlässigkeitsverlauf öffnen und passende Fehler suchen.
- Bei Speicherplatzproblemen Datenträgerbereinigung und große Benutzerordner prüfen.
- Bei Druckerproblemen Warteschlange, Treiber, Printserver und Testseite prüfen.
- Bei Sound- oder Displayproblemen Ausgabegerät, Skalierung, Auflösung und Treiber prüfen.

## Nützliche Befehle

```text
devmgmt.msc
eventvwr.msc
msinfo32
```
