# Windows Update

## Ziel

Kurze Checkliste für Windows Updates auf einem Client.

## Checkliste

- Windows Update öffnen und nach Updates suchen.
- Option für Updates anderer Microsoft-Produkte aktivieren.
- Option für Neustartbenachrichtigung aktivieren.
- Optionale Treiberupdates nur installieren, wenn sie wirklich benötigt werden.
- Nach der Installation einen Neustart durchführen, wenn Windows ihn anfordert.
- Nach dem Neustart erneut nach Updates suchen, bis keine wichtigen Updates mehr offen sind.
